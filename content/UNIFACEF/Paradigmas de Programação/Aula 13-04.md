Exercício Integrador: Sistema de Gestão de Logística  
**Objetivo:** Implementar um sistema que gerencie uma Empresa de transportes, que possui diversos Veiculos em sua frota.
1. Classe Abstrata: Veiculo (Herança + Abstrato)  
Esta será a superclasse da nossa hierarquia. Ela não pode ser instanciada.

- **Atributos Protegidos:** String placa, double capacidadeCarga.
- **Método Concreto:** public void exibirDados() (Exibe placa e capacidade).
- **Método Abstrato:** public abstract double calcularCustoViagem(double distancia).
- _Regra:_ Cada tipo de veículo tem um custo por km diferente.

2. Classes Filhas: Caminhao e Van (Polimorfismo)

- **Classe Caminhao**:
- Atributo próprio: int quantidadeEixos.
- **Anulação (@Override)**: O custo da viagem é: (distancia * 5.00) + (quantidadeEixos * 50.00).

**Classe Van**:

- Atributo próprio: boolean refrigerada.
- **Anulação (@Override)**: O custo da viagem é: distancia * 3.00. Se for refrigerada, adiciona uma taxa fixa de R$ 100,00.

3. Classe do Todo: EmpresaLogistica (Agregação)  
Esta classe irá gerenciar a frota.

- **Atributos:** String nomeEmpresa, ArrayList<Veiculo> frota.
-  **Método adicionarVeiculo(Veiculo v)**: Recebe o objeto pronto e guarda na lista (**Agregação**).
- **Método gerarRelatorioCustos(double distancia)**:
- Deve percorrer a lista de veículos e, para cada um, chamar o método calcularCustoViagem(distancia) (**Polimorfismo**).
- Deve exibir o custo individual de cada veículo da frota.

4. Classe de Teste: SistemaPrincipal  
No método main, o aluno deve: 
- Criar uma instância de EmpresaLogistica.
- Criar 2 Caminhões e 2 Vans com dados diferentes. 
- Adicionar todos os veículos à empresa. 
- Chamar o método de relatório para uma viagem de 300 km.
